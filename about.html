Progressive Web App

Progressive Web App (PWA) is a type of web app that could be installed as it's own application. PWA was introduced in 2016 and is supported by Chromium based web browsers such as Chrome, Brave, Opera, ect. PWA utilizes a powerful tool in java that is called service workers.

Service worker is a web enhancement tool that is often used to do custom request. Service worker lifecycle starts by installing it, then activating it, and finally disposing it once it became redundant. Service worker is used to bridge communications between client-server and store the data in a cache.
